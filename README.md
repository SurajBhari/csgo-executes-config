# csgo-executes-config

![CS:GO](https://img.shields.io/badge/CS:GO-Practice-F89406?style=flat&logo=counter-strike&logoColor=white)
![SourceMod](https://img.shields.io/badge/SourceMod-Executes-EA7500?style=flat)

An **updated, rebalanced set of [SourceMod Executes](https://github.com/splewis/csgo-practice-mode) spawn configs** for CS:GO practice/retake servers — the maintained successor to my original [configs](https://github.com/SurajBhari/configs) collection.

## What's improved over the defaults

1. **Fixes stuck spawns** on various maps (no more spawning into geometry).
2. **More fakes** — extra decoy/utility spawns for more realistic site reads.
3. **Better balanced** site-take / retake scenarios.

## Maps covered

Ancient, Cache, Cbble, Dust2 (+ night/old variants), Inferno, Mirage, Nuke, Overpass, and more — the competitive map pool.

## Install

Copy the configs into your server so they land at:

```
csgo/addons/sourcemod/configs/executes/<map>.cfg
```

Then reload the map (or the Executes plugin) to apply them.
